Alta personalizzazione in base alle esigenze del proprio blog. [Plugin Homepage](http://www.neednote.it "www.neednote.it")

== Description ==

Automaticamente aggiunge i pulsanti e i form di neednote ai posts.


== Installation ==

**Installazione**

1. Scarica il file `neednote.zip` ed estrai la cartella neednote. 
1. Fai l'upload dell'intera cartella neednote (non solo i file all'interno!) nella cartella `wp-contents/plugins`.

**Attivazione**

1. Dal pannello di amministrazione di Wordpress vai alla pagina plugins.
1. Attiva il plugin neednote,vedrai apparire un sottomenu neednote, dal quale potrai modificare le impostazioni.
